Lion School é uma aplicação web educativa para gerenciamento de cursos e alunos.  
Permite visualizar os cursos disponíveis, listar os alunos de cada curso e mostrar o desempenho individual de cada aluno em um gráfico interativo.

## Funcionalidades

- Listar cursos disponíveis
- Mostrar alunos de cada curso
- Visualizar perfil de cada aluno
- Gráfico de desempenho com valores e legendas
- Navegação entre telas (Tela de cursos, Tela de alunos, Tela de perfil)
- Botões de voltar e sair

## Tecnologias Utilizadas

- HTML5
- CSS3
- JavaScript (ES6)
- Fetch API para consumo de dados
